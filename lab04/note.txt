Czasami rozwiazanie problemu latwiejsze po transformacji. Potem wynik transformujemy z powrotem.

Transformacja Fouriera:

- funkcja bazowa (nieskonczenie wiele)
- f(x)=y=sum(y_i * f_i(x))  - dla danego x_i mamy y_i, inaczej 0 (funkcja ma wartosc dla argumentu)
- w transformacie zmieniamy przestrzen funkcji bazowych - kazdej z funkcji przypisujemy  cosinus
- funkcje roznia sie czestotliwoscia
- czestotliwosci przypisujemy amplitude
- tw. Fouriera (dla dowolnej ilosci pkt da sie tak zsumowac cosinusy zeby przez nie (punkty) przeszlo)
- dla punktow przypisujemy jasnosc

- y = A*sin(omega*x+phi)+stala
- omedze przypisujemy A i phi
- A \in [0, +inf)
- phi \in [0, 2pi)  albo w matlab \in [-pi, pi)

- obie wartosci mozna przedstawic za pomoca liczby zespolonej
- z = a + ib
- |z| = A, kat phi
- z = A*e^(i*phi)
- A = sqrt(a^2 + b^2)
- phi = arctg(b/a)

- widmo amplitudowe - najwieksze amplitudy w rogach - rozblyski (najmiejsze czestotliwosci)
- powtarza sie okresowo
